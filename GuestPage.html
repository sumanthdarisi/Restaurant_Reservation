New Reservation
This tab helps you to make a new table(s) reservation on particular day and time. You are requested to provide contact details(mobile and email ID). No need to provide any debit/ credit cards details here in this page.
Update Reservation
This tab helps user/guest to make changes to your previous reservation. You are requested to enter the "Unique confirmation code" of reservation which you want to update (this code was mailed already. Then user allowed to give changes (time/date/party size..) and save changes. Confirmation code may not change even after changes made.
Cancel Reservation
This tab helps user to cancel their reservation made in past. You are requested to enter the "unique confirmation code" of reservation which you want to cancel(this code was mailed already). This action can not be undo after hitting cancel my reservation button.
